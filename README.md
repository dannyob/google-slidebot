# google-slidebot

A Python CLI application

## Installation

```bash
# Development installation
make dev-setup

# Or manually:
uv venv
source .venv/bin/activate
uv pip install -e .
```

## Usage

```bash
google-slidebot --help
google-slidebot hello
```

## Development

```bash
make help          # Show all commands
make test          # Run tests
make check         # Lint + format + test
```
